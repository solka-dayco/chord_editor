Chorditor

코드 에디터 (메인화면)

  • 만들고 싶은 프랫에 Dot 찍고 뮤트 적용하기.
  • B 버튼으로 바레 적용
  • 프랫번호로 위치 조정 가능
  • 재생 버튼으로 들어 볼 수 있음
  • 코드 이름은 직접 선택해야 함. (우측에 추천 코드명이 나타남. 복잡한 코드의 경우는 결과가 뜨지 않을 수 있음)
  • 갤러리에 직접 저장하거나 프로젝트에 모아둘 수 있음

프로젝트 페이지

  • 왼쪽 사이드바를 열고 + 버튼으로 프로젝트 생성
  • 코드 슬롯에 모아뒀던 코드를 드래그해서 삽입. 아래 텍스트 입력란엔 메모, 가사 등 원하는 텍스트를 작성.
  • ⚠️ 주의: 현재 클립보드를 통한 붙여넣기 버그가 해결되지 않음. 반드시 꾹 눌러서 직접 붙여넣기로 붙여넣어야 정상적으로 텍스트가 적용됨.
  • BPM, 카포, 메트로놈 on/off 가능.
  • 재생을 통해서 처음부터 BPM에 맞게 코드가 재생됨. (4칸 모드 : 코드슬롯 당 1/2마디, 8칸 모드 : 코드슬롯 당 1/4마디) 중간에 코드 터치 시 그 지점부터 재생 가능.
  • 공유 버튼으로 제목과 텍스를 제외한 모든 데이터 공유가능. (저작권 문제로 텍스트 공유는 안되도록 했습니다ㅎㅎ...)
