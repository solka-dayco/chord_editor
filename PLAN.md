# PLAN.md — Chord Editor 프로젝트 계획서

## 1. 프로젝트 개요

기타 코드표를 웹 브라우저 및 모바일 앱에서 직접 제작하고 PNG로 저장할 수 있는 툴.
유튜브 기타 강사, 악보 제작자, 기타 학습자를 주요 타겟으로 한다.

---

## 2. 핵심 기능

| 기능 | 설명 |
|------|------|
| 코드명 빌더 | 근음 · 3화음 · 7음 · 기능 · 텐션 · 분수코드 조합으로 코드명 자동 생성 |
| 코드표 편집 | 캔버스 클릭으로 dot · 오픈 · 뮤트 입력. 한 줄에 하나의 음만 허용 |
| 바레 표시 | 같은 프렛에 dot 2개 이상 시 바레 버튼 활성화, 클릭으로 바레 이미지 전환 |
| 근음 표시 | 근음 모드 ON 시 가장 아래 줄 dot/오픈을 빨간색으로 자동 표시 |
| 손가락 번호 모드 | 번호 모드 ON/OFF. OFF 시 텍스트 없는 원형 이미지로 표시 |
| 프렛 번호 | 2번째 프렛 하단에 프렛 번호 표기 (2~18 입력 가능) |
| 코드 재생 | Karplus-Strong 합성음으로 스트럼 방식 재생 (6번줄→1번줄, 75ms 간격) |
| PNG 저장 | 웹: 680px×3 배율 다운로드 / 앱: 갤러리 직접 저장 (Chorditor 앨범) |
| 반응형 캔버스 | 화면 너비에 따라 캔버스 크기 자동 조정, PNG는 항상 원본 해상도 유지 |
| 모바일 앱 | Capacitor 기반 Android APK 빌드 지원 |

## 3. 현재까지 구현된 기능

- [x] 코드명 빌더 (근음 12음계 #/b 모드, 3화음, 7음, 기능, 텐션 다중선택, 분수코드)
- [x] 캔버스 기반 코드표 (6줄 × 4프렛, 너트 포함)
- [x] 커스텀 이미지 dot (root1~4, common1~4, root_t, common_t)
- [x] 커스텀 이미지 오픈/뮤트/오픈루트 (open, mute, open_root)
- [x] 커스텀 이미지 바레 (barre_two ~ barre_six)
- [x] 오픈/뮤트 순환 클릭 방식 (open ↔ mute)
- [x] 한 줄에 하나의 음만 허용 (dot/오픈/뮤트 상호 배타)
- [x] 바레 버튼 (HTML overlay, PNG에 미포함)
- [x] 근음 모드 토글 (rootIndex 기반 분리 설계, ON/OFF 완전 복구)
- [x] 손가락 번호 모드 토글 (OFF 시 open/open_root 이미지 사용)
- [x] 프렛 번호 입력 (2~18, 유효성 검사 포함)
- [x] 코드명 캔버스 렌더링 (텐션 위첨자 처리)
- [x] Karplus-Strong 합성음 재생 (OfflineAudioContext 기반)
- [x] 스트럼 재생 (6번줄→1번줄, 75ms 간격)
- [x] 재생 중단 후 재시작 시 이전 소리 stop 처리
- [x] 음 계산 (개방현 미디 + 실제 프렛 번호 기반)
- [x] PNG 고화질 저장 (3× 배율, 680px 기준)
- [x] 반응형 캔버스 (window resize 대응)
- [x] GitHub Pages 배포
- [x] Pretendard 폰트 적용
- [x] 모바일 UI 대응
- [x] Capacitor Android APK 빌드
- [x] Android 갤러리 직접 저장 (@capacitor-community/media)
